📋 ภาพรวมระบบ

BMS Training เป็นระบบบริหารจัดการการอบรมสำหรับบุคลากรทางการแพทย์ ครอบคลุมตั้งแต่การลงทะเบียน เช็คชื่อ ทำแบบทดสอบ ประเมินความพึงพอใจ ไปจนถึงการพิมพ์เอกสาร

📝
ลงทะเบียนอบรม
ผู้เข้าอบรมลงทะเบียนผ่านเว็บ เลือกหลักสูตรและรอบ กรอกข้อมูลส่วนตัว รับ QR Code ยืนยัน
📲
เช็คชื่อ QR Code
เช็คชื่อด้วยการสแกน QR Code หรือค้นหาชื่อจากรายการ รองรับ Walk-in
✏️
แบบทดสอบออนไลน์
ทำแบบทดสอบออนไลน์ ดูผลทันที รับใบประกาศนียบัตรอัตโนมัติ
📊
Dashboard ผลประเมิน
วิเคราะห์ผลความพึงพอใจแบบ Real-time พร้อมกราฟ Radar, Heatmap, Trend

👥 ประเภทผู้ใช้งาน

ประเภทสิทธิ์วิธีเข้าถึง
ผู้เข้าอบรม Public ลงทะเบียน, ทำแบบทดสอบ, แบบประเมิน เปิดเว็บได้เลย ไม่ต้อง Login
เจ้าหน้าที่ / Admin Admin จัดการรอบ, ดูรายชื่อ, เช็คชื่อ, พิมพ์เอกสาร กด Admin ที่มุมขวาบน → ใส่ Username / Password
Super Admin Super ทุกอย่าง รวมถึงจัดการสาขา, ผู้ใช้งาน Login ด้วย account ระดับ Super Admin

🔗 การเข้าใช้งานตามสาขา

ระบบรองรับหลายสาขาในลิงก์เดียวกัน โดยใช้ parameter ?site= ใน URL

💡
ตัวอย่าง URL ตามสาขา
https://yoursite.com/?site=theme_1 → สาขา 1
https://yoursite.com/?site=theme_2 → สาขา 2
https://yoursite.com/ → แสดงทุกสาขา (ไม่กรอง)

เมื่อกำหนด ?site= ระบบจะกรองแสดงเฉพาะหลักสูตร รอบอบรม และผู้ลงทะเบียนของสาขานั้นโดยอัตโนมัติ รวมถึงดึงชื่อสถานพยาบาลมากรอกในเอกสารอัตโนมัติ

📲 ติดตั้งเป็นแอป (PWA)

ระบบรองรับการติดตั้งลงหน้าจอโฮมของมือถือ/แท็บเล็ต เหมือนแอปทั่วไป เปิดใช้งานได้เร็วขึ้นและใช้งานได้บางส่วนแม้ไม่มีอินเทอร์เน็ต

💡
เงื่อนไขการแสดงปุ่มติดตั้ง
ปุ่ม/แบนเนอร์ "ติดตั้งแอป" จะปรากฏ เฉพาะหลัง Login เข้าระบบ Admin แล้วเท่านั้น และจะเตือนซ้ำทุกครั้งที่ Login หากเครื่องนั้นยังไม่เคยติดตั้ง

วิธีติดตั้ง

อุปกรณ์วิธีติดตั้ง
Android (Chrome)กดปุ่ม "ติดตั้งแอป" มุมขวาบน หรือแบนเนอร์ด้านล่างจอ → กดยืนยันในป๊อปอัพของเบราว์เซอร์
iOS (Safari)กดปุ่ม "ติดตั้งแอป" → ทำตามคำแนะนำ: แตะปุ่ม แชร์ (Share) แถบด้านล่าง → เลือก "เพิ่มไปยังหน้าจอโฮม"
Desktop / อื่นๆกดปุ่ม "ติดตั้งแอป" → ทำตามคำแนะนำแตะเมนู ⋮ ของเบราว์เซอร์ → เลือก "ติดตั้งแอป"

การใช้งานออฟไลน์และอัปเดต

  • หากอินเทอร์เน็ตหลุด ระบบจะแสดงแถบแจ้งเตือน "ออฟไลน์" ด้านบน และซิงค์ข้อมูลอัตโนมัติทันทีที่กลับมาออนไลน์
  • เมื่อระบบมีเวอร์ชันใหม่ จะมีแถบแจ้งเตือน "มีเวอร์ชันใหม่ของระบบ" ให้กด "อัปเดตเลย" เพื่อโหลดเวอร์ชันล่าสุด

📝 ลงทะเบียนอบรม

ผู้เข้าอบรมสามารถลงทะเบียนได้เองผ่านหน้าเว็บโดยไม่ต้อง Login

1
เลือกหลักสูตร
หน้าแรกจะแสดงการ์ดหลักสูตรทั้งหมด สามารถ ค้นหา หรือกรองตามสถานะ (มีที่ว่าง / ใกล้เต็ม / เต็มแล้ว) ได้
2
เลือกรอบอบรม
กดการ์ดหลักสูตรที่ต้องการ ระบบจะแสดงรอบอบรมที่มีอยู่ พร้อมวันที่ เวลา สถานที่ และจำนวนที่นั่งคงเหลือ
3
กรอกข้อมูลส่วนตัว
กรอก คำนำหน้า, ชื่อ-นามสกุล, ตำแหน่ง, แผนก, เบอร์โทร แล้วกด "ลงทะเบียน"
4
รับ QR Code
เมื่อลงทะเบียนสำเร็จ ระบบแสดง QR Code ของผู้เข้าอบรม — บันทึกภาพหรือ screenshot ไว้ใช้เช็คชื่อในวันอบรม
⚠️
หมายเหตุ
หากรอบอบรมเต็มแล้ว ปุ่ม "ลงทะเบียน" จะถูกปิด ไม่สามารถลงทะเบียนได้

📲 เช็คชื่อเข้าอบรม

หน้าเช็คชื่อสำหรับเจ้าหน้าที่ Login แล้ว ใช้ได้ 3 วิธี

วิธีที่ 1 — สแกน QR Code

1
กด "เปิดกล้องสแกน"
ระบบจะขอสิทธิ์ใช้กล้อง กด "อนุญาต" ในป๊อปอัพของเบราว์เซอร์
2
ส่องกล้องที่ QR Code
นำ QR Code ของผู้เข้าอบรมเข้าในกรอบสแกน ระบบจะบันทึกเช็คชื่อทันที
3
ดูผลการเช็คชื่อ
แสดงชื่อ-นามสกุล ประเภท รอบอบรม และสถานะ "เช็คชื่อแล้ว" หรือ "ไม่พบ"

วิธีที่ 2 — พิมพ์ชื่อด้วยมือ

กดปุ่ม "พิมพ์ชื่อ" → ค้นหาชื่อผู้เข้าอบรม → กดเลือกเพื่อบันทึกเช็คชื่อ

วิธีที่ 3 — รายชื่อ (List)

กดแท็บ "รายชื่อ" → เลือกรอบอบรม → ติ๊กถูกหน้าชื่อผู้ที่มาอบรม

Walk-in

กดปุ่ม "Walk-in" (สีม่วง) สำหรับผู้ที่ยังไม่ได้ลงทะเบียนล่วงหน้า — กรอกข้อมูลแล้วกด "บันทึก" ระบบจะเพิ่มรายชื่อและเช็คชื่อพร้อมกัน

📊 แบบประเมินความพึงพอใจ

ผู้เข้าอบรมกรอกแบบประเมินหลังจบการอบรมผ่านหน้า survey.html?site=xxx

  • ประเมิน 6 ด้าน ได้แก่ หลักสูตร, วิทยากร, สื่อการสอน, สถานที่, การจัดการ และประโยชน์ที่ได้รับ
  • คะแนนแต่ละด้าน 1–5 คะแนน
  • มีช่องข้อเสนอแนะเพิ่มเติม
  • ผู้ Admin ดูสรุปผลได้ที่แท็บ ผลประเมิน

✏️ ทำแบบทดสอบ (ผู้เข้าอบรม)

เข้าหน้าแบบทดสอบได้ที่ /quiz/ หรือผ่านลิงก์ที่ผู้ดูแลระบบแชร์ให้

1
ลงทะเบียน / เข้าสู่ระบบ
กรอกชื่อ อีเมล แผนก และหน่วยงาน (ใช้ครั้งแรก) หรือกรอกอีเมลเพื่อเข้าสู่ระบบถ้าเคยสอบแล้ว
2
เลือกหลักสูตร
เลือกหลักสูตรที่ต้องการสอบ ระบบแสดงจำนวนข้อ เวลา และเกณฑ์ผ่าน
3
ทำแบบทดสอบ
ตอบคำถามทีละข้อ มีตัวเลือก 3–5 ตัว กดถัดไป / ย้อนกลับได้ มีนับถอยหลังเวลา
4
ดูผลและรับใบประกาศ
แสดงคะแนน % และสถานะ ผ่าน/ไม่ผ่าน ถ้าผ่านสามารถดาวน์โหลดใบประกาศนียบัตร PDF หรือส่งทางอีเมลได้
🎓
ใบประกาศนียบัตร
สามารถดูประวัติการสอบและดาวน์โหลดใบประกาศย้อนหลังได้ที่หน้า "ประวัติการสอบ"

🔐 เข้าสู่ระบบ Admin

1
กดปุ่ม "Admin" มุมขวาบน
หน้า Login จะปรากฏขึ้น
2
กรอก Username และ Password
ใช้ account ที่ผู้ดูแลระบบสร้างให้ — ติ๊ก "จดจำการเข้าสู่ระบบ" เพื่อให้เบราว์เซอร์นี้จำ Username/Password ไว้ ไม่ต้องกรอกใหม่ครั้งหน้า
3
เข้าสู่หน้า Admin Panel
ระบบแสดง Tab ต่างๆ ด้านบน ได้แก่ รอบอบรม, ประเภท, ข้อมูลพื้นฐาน, ผู้ลงทะเบียน ฯลฯ
⚠️
ออกจากระบบ
กดปุ่ม "ออกจากระบบ" ทุกครั้งหลังใช้งานเสร็จ โดยเฉพาะบนคอมพิวเตอร์สาธารณะ (หากติ๊ก "จดจำการเข้าสู่ระบบ" ไว้ ควรระวังเป็นพิเศษบนเครื่องที่ใช้ร่วมกับผู้อื่น)

📅 จัดการรอบอบรม

แท็บ รอบอบรม — จัดการกำหนดการอบรมทั้งหมด

ดูรายการรอบอบรม

ตารางแสดง ประเภท, รอบ, วันที่, เวลา, สถานที่, วิทยากร, จำนวนลงทะเบียน/ทั้งหมด, จำนวนเข้าอบรมจริง และสถานะ สามารถกรองตามประเภทได้ที่ dropdown ด้านบน

เพิ่มรอบอบรม

1
กด "+ เพิ่มรอบ"
หน้าต่างเพิ่มรอบจะปรากฏ
2
กรอกข้อมูลรอบอบรม
เลือกประเภทอบรม, กรอกชื่อรอบ, วันที่, เวลาเริ่ม-สิ้นสุด, สถานที่, วิทยากร, จำนวนที่นั่ง
3
กด "บันทึก"
รอบอบรมใหม่จะปรากฏในตารางและหน้าลงทะเบียนทันที

แก้ไข / ลบรอบอบรม

กดไอคอน ✏️ เพื่อแก้ไข หรือ 🗑️ เพื่อลบ ในคอลัมน์ จัดการ

🚫
ข้อควรระวัง
การลบรอบอบรมที่มีผู้ลงทะเบียนแล้ว จะลบข้อมูลการลงทะเบียนทั้งหมดในรอบนั้นด้วย

🗂️ จัดการประเภทอบรม

แท็บ ประเภท — จัดการหมวดหมู่ / ประเภทของการอบรม

ตารางแสดงชื่อประเภท, คำอธิบาย, จำนวนรอบ, จำนวนผู้ลงทะเบียน และว่ามีแบบทดสอบผูกอยู่หรือไม่

เพิ่มประเภทอบรม

กด "+ เพิ่มประเภท" → กรอกชื่อประเภทและคำอธิบาย → เลือกแบบทดสอบที่ผูกกัน (ถ้ามี) → กด "บันทึก"

💡
ผูกประเภทกับแบบทดสอบ
สามารถเลือกหลักสูตรแบบทดสอบ (Quiz) ที่ผูกกับประเภทอบรมนี้ได้ เมื่อผู้เข้าอบรมผ่านการเช็คชื่อ ระบบจะส่งลิงก์แบบทดสอบให้อัตโนมัติ

📌 ข้อมูลพื้นฐาน

แท็บ ข้อมูลพื้นฐาน — จัดการรายการ Dropdown ที่ใช้ในระบบ

รายการใช้ที่
🎤 วิทยากรDropdown ในฟอร์มเพิ่มรอบอบรม
📍 สถานที่Dropdown ในฟอร์มเพิ่มรอบอบรม
🏢 แผนกDropdown ในฟอร์มลงทะเบียน และ Walk-in
🪪 คำนำหน้าชื่อDropdown ในฟอร์มลงทะเบียน

วิธีเพิ่ม: พิมพ์ในช่องด้านล่างของแต่ละรายการ แล้วกด + หรือกด Enter

วิธีลบ: กดไอคอน 🗑️ ที่หน้าแต่ละรายการ

👤 ผู้ลงทะเบียน

แท็บ ผู้ลงทะเบียน — ดูและจัดการรายชื่อผู้ที่ลงทะเบียนทั้งหมด

ค้นหาและกรอง

  • ช่องค้นหา: ค้นหาชื่อ, นามสกุล, แผนก
  • Dropdown ประเภท: กรองตามประเภทอบรม
  • Dropdown รอบ: กรองตามรอบอบรม

เพิ่มรายการด้วยมือ

กด "+ เพิ่มรายการ" → กรอกข้อมูลผู้เข้าอบรม → เลือกประเภทและรอบ → กด "บันทึก"

Import จาก Excel

กดไอคอน Import → ดาวน์โหลด Template → กรอกข้อมูล → อัปโหลดไฟล์ Excel ระบบจะ Import รายชื่อทั้งหมด

พิมพ์แบบฟอร์มเซ็นชื่อ

กดปุ่ม "พิมพ์แบบฟอร์ม" → กรอกข้อมูลหัวข้อเอกสารและผู้ลงนาม (ชื่อสถานพยาบาลจะดึงจากสาขาอัตโนมัติ) → กด "พิมพ์"

ดู QR Code

กดไอคอน QR ในคอลัมน์ QR เพื่อดู QR Code ของผู้ลงทะเบียนแต่ละคน

⚠️
เคียร์ตามสาขา
ปุ่ม "เคียร์ตามสาขา" จะลบข้อมูลการลงทะเบียนทั้งหมดของสาขาที่เลือก — ใช้ด้วยความระมัดระวัง

🗄️ ตรวจสอบคีย์ยอดตั้งต้น

แท็บ ตรวจสอบคีย์ยอด — ติดตามความคืบหน้าการคีย์ยอดสต๊อกตั้งต้นในระบบ BMS-INVENTORY แยกรายแผนกของสาขาที่ Login อยู่

สรุปความคืบหน้า

การ์ดสถิติด้านบนแสดง แผนกทั้งหมด, คีย์ยอดแล้ว, ยังไม่คีย์ และ % ความคืบหน้า พร้อมแถบความคืบหน้า (Progress bar)

อัปเดตสถานะแผนก

ในตาราง ปรับ Dropdown สถานะของแต่ละแผนกเป็น "คีย์ยอดแล้ว" หรือ "ยังไม่คีย์" — ระบบจะบันทึกวันที่/เวลาให้อัตโนมัติ หากแผนกใดมีเหตุผลที่ยังไม่คีย์ (เช่น ปิดหน่วยงานชั่วคราว) สามารถกรอกในช่อง "สาเหตุที่ยังไม่คีย์" ได้

ค้นหาและสรุปเป็นภาพ

  • ใช้ช่องค้นหาด้านบนเพื่อกรองรายชื่อแผนก
  • กด "รีโหลด" เพื่อดึงข้อมูลล่าสุด
  • กด "สรุปเป็นภาพ" เพื่อบันทึกสรุปความคืบหน้าเป็นรูปภาพ ส่งต่อให้ทีมงานได้สะดวก

📈 Dashboard ผลการประเมิน

แท็บ ผลประเมิน — วิเคราะห์ผลความพึงพอใจจากผู้เข้าอบรม

ตัวกรอง

  • สาขา — เลือกดูผลของสาขาใดสาขาหนึ่ง
  • รอบอบรม — กรองตามรอบที่ต้องการ หรือดูทุกรอบ
  • ช่วงเวลา — 7 / 30 / 90 วันล่าสุด หรือทั้งหมด

กราฟและข้อมูลที่แสดง

ส่วนรายละเอียด
KPI Cardsจำนวนผู้ประเมิน, คะแนนเฉลี่ย, % ผู้ที่ต้องการอบรมเพิ่ม
Radar Chartภาพรวม 6 ด้านในรูปแบบแผนภูมิใยแมงมุม
Bar Chartคะแนนเฉลี่ยแต่ละด้าน (เต็ม 5.00)
Distribution Chartการกระจายระดับคะแนน (%)
Heatmapคะแนนรายข้อทุกด้าน (สีแสดงระดับ)
Trend Chartแนวโน้มคะแนนแต่ละด้านตามรอบอบรม
ข้อเสนอแนะรายการ Comment จากผู้ประเมิน

🖨️ พิมพ์เอกสาร

แท็บ พิมพ์เอกสาร — สร้างและพิมพ์ใบเซ็นชื่อผู้เข้าอบรม

ตั้งค่าเอกสาร

หัวข้อเอกสาร

  • เลือกจาก Dropdown รายการที่บันทึกไว้ หรือพิมพ์หัวข้อเองในช่อง Textarea
  • กด "จัดการหัวข้อ" เพื่อเพิ่ม/แก้ไข/ลบ รายการ Dropdown

ข้อมูลเอกสาร

ฟิลด์หมายเหตุ
สถานพยาบาลเลือกสาขา — ชื่อจะถูกดึงอัตโนมัติ
วันที่วันที่ในเอกสาร
จำนวนแถวเลือกจากตัวเลือก 25 / 50 / 75 / 100 แถว (แต่ละแถวคือ 1 ช่องเซ็นชื่อ)
แสดงช่องเจ้าหน้าที่เพิ่มคอลัมน์ชื่อเจ้าหน้าที่ BMS

ผู้ลงนาม

  • ซ้าย (BMS) — ชื่อ, ตำแหน่งของผู้แทน BMS
  • ขวา (สถานพยาบาล) — ชื่อ, ตำแหน่ง, ชื่อโรงพยาบาล (ดึงจากสาขาอัตโนมัติ)
  • กด "บันทึกเป็นค่าเริ่มต้น" เพื่อจำค่าผู้ลงนามไว้ใช้ครั้งหน้า

พิมพ์

ด้านขวาแสดง ตัวอย่างเอกสาร แบบ Real-time กด "พิมพ์เอกสาร" มุมขวาบนเพื่อสั่งพิมพ์ออกเครื่องพิมพ์ รองรับกระดาษ A4

💡
หลายหน้า
หากมีแถวมากกว่า 1 หน้ากระดาษ ระบบจะแสดงหมายเลขหน้า หน้า 1/2, หน้า 2/2 โดยอัตโนมัติ หากมีเพียง 1 หน้า จะไม่แสดงหมายเลขหน้า

พิมพ์แบบฟอร์มจาก Tab ผู้ลงทะเบียน

ใน Tab ผู้ลงทะเบียน → กดปุ่ม "พิมพ์แบบฟอร์ม" → กรอกข้อมูลในหน้าต่างที่เปิดขึ้น ชื่อสถานพยาบาลจะถูกดึงจากสาขาปัจจุบันอัตโนมัติ

สามารถกด "จำค่าเริ่มต้นผู้ลงนาม" เพื่อบันทึกชื่อ/ตำแหน่งผู้ลงนามฝั่ง BMS และฝั่งโรงพยาบาลไว้ใช้ครั้งถัดไป โดยจำแยกเป็นรายสาขา
